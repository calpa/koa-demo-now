# Demo of Koa.js

[Live demo](https://koa-demo-i2qhrak4b.now.sh/)

## Installation

```bash
git clone https://github.com/calpa/koa-demo-now.git
cd koa-demo-now
npm install
npm start
```

## Development

Add `app.listen(3000)` to `index.js`, and open localhost:3000
